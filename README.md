# MAZDOOR
<img align="center" alt="Coding" width="850" src="images/1877.webp"></img><br><br><br>
As we know that nowadays we daily need some minute work done sometimes those are like 
plumbing, electricians, etc. For most of the times we are not able to get workers for job to done. 
So, we have to wait for some time while problem is still there. So that we are now introducing to 
are new project that is based on website which will hire daily wage people or some pro workers 
for your work which will get via bid. Project will not provide services it will help worker and client 
get connected to each other and fulfill their needs. This project will make task easier for both client 
and worker. 
# Aim 
To make task Easier for client and worker, by creating the job opportunity and bringing client and 
worker together on single platform.
